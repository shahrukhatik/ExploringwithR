NC Birth Weight Data

Shahrukh Khan May 6th, 2016

Brief Description

The data set contains 8 variables measured on 2000 infants born in North Carolina. The variables of the dataset are as follows:

  • Patient.ID
  • Mother Minority
  • Mother Age
  • Marital Status
  • Mother Smoker
  • Gender
  • Number of Prenatal Visits
  • Birth Weight Grams
